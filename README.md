# minify web front

with this project you can minify your html, css, js files

## instruction

create folder data
```sh
cp data
```

copy your html, css, js files to this folder
```sh
cp {your files} data/
```

minify your html, css, js files
```sh
npm run build
```

check are they working
```sh
npm run start
```
open in browser: http://localhost:8080
